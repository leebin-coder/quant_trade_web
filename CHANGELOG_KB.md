# 知识库功能更新日志

## v1.1 - 2025-01-08

### 界面优化

#### 圆角减小
- **面板圆角**: 12px → 8px (更精致)
- **按钮圆角**: 6px → 4px
- **树节点圆角**: 5px → 4px
- 整体更加紧凑和现代

#### 按钮优化
- **关闭按钮悬浮效果**: 现在悬浮时显示红色背景 (#ff3b30)
  - 使用 `!important` 确保样式生效
  - 提供更明确的视觉反馈
  - 符合macOS的设计规范

#### 侧边栏优化
- **移除"目录"标题**: 更简洁的设计
- **控制按钮改用图标**:
  - 展开: `<ExpandOutlined />`
  - 收起: `<ShrinkOutlined />`
  - 定位: `<AimOutlined />`
- **按钮尺寸**: 24x24px 正方形
- **只显示图标**: 无文字,更节省空间
- **侧边栏头部高度**: 36px → 32px
- **按钮布局**: 右对齐,紧凑排列

#### 布局调整
- **内容展示区**: 完全铺满,无额外间隙
- 左侧树形导航和右侧内容区域无缝衔接
- 顶部工具栏与内容区域紧密贴合

### 技术细节

#### 新增图标导入
```jsx
import {
  ExpandOutlined,   // 展开所有
  ShrinkOutlined,   // 收起所有
  AimOutlined,      // 定位当前
} from '@ant-design/icons'
```

#### CSS样式调整
```css
/* 面板圆角 */
border-radius: 8px;  /* 原12px */

/* 按钮圆角 */
border-radius: 4px;  /* 原6px */

/* 关闭按钮悬浮 */
.action-btn.close-btn:hover {
  background: #ff3b30 !important;
  color: #ffffff !important;
}

/* 侧边栏按钮 */
.sidebar-action-btn {
  width: 24px !important;
  height: 24px !important;
  /* 只显示图标 */
}
```

### 界面截图对比

#### 圆角变化
- 之前: 较大圆角 (12px/6px/5px)
- 现在: 精简圆角 (8px/4px/4px)
- 效果: 更紧凑、更现代

#### 侧边栏变化
- 之前: "目录" + 文字按钮 (展开/收起/定位)
- 现在: 纯图标按钮 (▣/▢/⊙)
- 效果: 更简洁、节省空间

#### 关闭按钮
- 之前: 悬浮时浅灰色背景
- 现在: 悬浮时鲜红色背景
- 效果: 更醒目、更符合操作直觉

### 用户体验提升

1. **视觉一致性**: 所有圆角统一减小,整体风格更统一
2. **操作直观性**: 关闭按钮红色提示,防止误操作
3. **空间利用**: 侧边栏按钮图标化,节省水平空间
4. **无缝体验**: 内容区域铺满,视觉更连贯

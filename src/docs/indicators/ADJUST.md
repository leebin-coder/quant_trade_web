# 复权处理

复权是对股票价格和成交量进行权息修复，以保持股票价格走势的连续性。

## 复权类型

### 不复权
保持股票的实际交易价格，不做任何调整。

**特点**：
- 显示真实的历史交易价格
- 价格会因除权除息出现跳空缺口
- 适合查看股票的真实成交价格

### 前复权
以目前价格为基准，保持现有价格不变，将历史价格向下调整。

**特点**：
- 当前价格 = 实际价格
- 历史价格根据除权除息进行调整
- 适合分析当前价格水平下的历史走势

**计算公式**：
```
前复权价 = (不复权价 - 每股派息) ÷ (1 + 每股送股 + 每股配股)
```

### 后复权
以上市首日价格为基准，保持首日价格不变，将之后价格向上调整。

**特点**：
- 首日价格 = 实际价格
- 之后价格根据除权除息进行调整
- 适合分析股票上市以来的累计涨幅

**计算公式**：
```
后复权价 = 不复权价 × (1 + 每股送股 + 每股配股) + 每股派息
```

## 使用建议

1. **技术分析**：建议使用前复权，保持价格连续性
2. **查看真实价格**：使用不复权
3. **计算累计收益**：使用后复权
